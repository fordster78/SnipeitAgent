# SnipeitAgent
A windows based Snipeit Agent that will add system details to the Snipe-it asset Database

### Prerequisites

```
A Working Install of Snipe IT V4+
```

### Library Packages

```
Uses the Snipesharp c# library. All credit goes to barrycarey for his work.
```
